An Abstract Message Manager.

## CocoaPods Install

```ruby
pod 'RFMessageManager'
```

Specify develop branch to install the lastest version:

```ruby
pod 'RFMessageManager',
    :git => 'https://github.com/RFUI/RFMessageManager.git',
    :branch => 'develop',
    :subspecs => ['Manager', 'SVProgressHUD']
```
